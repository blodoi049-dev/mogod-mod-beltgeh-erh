  <h1>Булган аймгийн Могод суманд мод бэлтгэх эрх буюу гоожин авах заавар</h1>

  <p><strong>Нэгдүгээрт:</strong> Та өөрийн харьяа сумын Засаг даргын Тамгын газарт өөрийн биеэр очиж Байгаль хамгаалагчид хандана.</p>

  <p><strong>Хоёрдугаарт:</strong> Та хэдэн оны хэдэн сарын хэдэнд мод бэлтгэх, ямар маркын авто машинаар мод бэлтгэх /тухайн авто машин хэнийх болох, эзэмшигчийн овог нэр/, авто машины улсын дугаарыг мэдэгдэнэ.</p>

  <p><strong>Гуравдугаарт:</strong> Байцаагчийн өгсөн зааврын дагуу тухайн сумын тооцооны төв (банк)-д очиж зохих хэмжээний төлбөрийг төлнө.</p>

  <p><strong>Дөрөвдүгээрт:</strong> Төлбөр төлсөн баримтаа байцаагчид өөрийн биеэр авчирч өгснөөр та мод бэлтгэх нэг удаагийн эрхтэй болно.</p>

  <p><strong>САНАМЖ:</strong> Тухайн өдөр өөр улсын дугаартай, өөр маркын авто машинаар мод бэлтгэсэн тохиолдолд та болон эрх олгосон байцаагч хариуцлага хүлээхийг анхаарна уу.</p>

  <h2>Та гэр бүлээ хайрладаг шиг байгаль дэлхийгээ хайрлаарай. Танд амжилт хүсье.</h2>
